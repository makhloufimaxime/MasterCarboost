Projet d'applicatio N-Tiers

BASE DE DONNEES
La database est une database MySQL nommée Carboost
Importer celle-ci dans MYSQL et lancer dans le localhost
(ndc: root ; mdp: root)

BACKEND
Installer Node.js
Télécharger les modules de l'application ($ npm install dans le dossier api)
Lancer le serveur avec ($ node server.js)

FRONTEND
Ouvrir index.html dans votre navigateur
